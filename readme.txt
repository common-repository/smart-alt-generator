== Description ==

The Smart Alt Generator plugin revolutionizes image SEO and accessibility on WordPress sites using AI technology. It automatically generates SEO-friendly alt text for images, boosts visibility in search engines, and enhances accessibility in 11 different languages: English, Spanish, Chinese, Hindi, Arabic, French, Russian, Portuguese, German, Japanese, and Italian. The plugin is easy to use, integrates with the WordPress media library, and operates with minimal setup. The basic version allows for recognition and auto-generation of alt text for up to 50 images. Register for an API key from [External Service](https://smart-alt.com/) to connect and start optimizing your site's images for SEO and accessibility.

== Installation ==

1. Upload the `smart-alt-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Register and obtain an API key from [External Service](https://smart-alt.com/).
4. Enter your API key in the Smart Alt Generator settings in your WordPress dashboard.

== Frequently Asked Questions ==

= How do I obtain an API key? =
Register at [External Service](https://smart-alt.com/) to generate an API key in your account settings.

= Is the API key required? =
Yes, it connects your site with the service for alt text generation.

= Is Smart Alt Generator free? =
The basic version is free and allows for alt text generation for up to 50 images. Premium features may require a subscription.
